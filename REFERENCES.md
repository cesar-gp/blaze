# Links to references mentioned in the documentation

This document includes links to every reference manual or external page that is mentioned in the documentation of the Operating System. The list is sorted in alphabetical order.

## List of links

- [`FAT32`](http://download.microsoft.com/download/1/6/1/161ba512-40e2-4cc9-843a-923143f3456c/fatgen103.doc): Microsoft Extensible Firmware Initiative FAT32 File System Specification.
- [`INTEL64IA32`](https://cdrdv2.intel.com/v1/dl/getContent/671200): Intel® 64 and IA-32 Architectures Software Developer’s Manual Combined Volumes: 1, 2A, 2B, 2C, 2D, 3A, 3B, 3C, 3D, and 4.
- [`NASM301`](https://www.nasm.us/pub/nasm/releasebuilds/3.01/doc/nasmdoc.pdf): NASM – The Netwide Assembler (version 3.01).
- [`RBIL`](https://www.cs.cmu.edu/~ralf/files.html): Ralf Brown's Interrupt List. All files extracted and *INTERRUP.\** and *PORTS.\** concatenated.
